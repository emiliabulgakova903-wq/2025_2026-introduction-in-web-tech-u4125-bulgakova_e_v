Отчет по развертыванию персонального портфолио на MkDocs

Цель работы: Создание и кастомизация статического сайта-портфолио с использованием фреймворка MkDocs и темы Material, а также публикация проекта на GitHub Pages.

1. Подготовка и инициализация
Проект инициализирован командой mkdocs new . внутри локального репозитория. В качестве базовой темы выбрана Material for MkDocs, так как она предоставляет адаптивный интерфейс и поддержку темной темы.

![git_1](screenshots/sc_1.png)

![git_2](screenshots/sc_2.png)

2. Настройка конфигурации (mkdocs.yml)
В файле конфигурации были определены:

Навигация: Структурированы разделы «Главная», «Проекты» и «Контакты».

Цветовая схема: Настроено автоматическое переключение между светлой и темной темами.

Материалы проекта: Подключена папка assets для хранения презентаций и PDF-документов.

![git_3](screenshots/sc_3.png)

3. Техническая кастомизация (CSS)
Для создания уникального дизайна был создан файл docs/stylesheets/extra.css. В ходе работы были решены следующие задачи:

Интеграция фонового изображения: Использован метод фиксированной подложки с градиентной заливкой (linear-gradient) для сохранения читаемости текста.

Работа со слоями (Z-index): Исправлена ошибка перекрытия интерактивных элементов (правого меню TOC) фоновым слоем.

Стилизация элементов: Добавлено скругление для фото профиля и эффект «стеклянной морфизма» (прозрачности) для контентных блоков.

![git_4](screenshots/sc_4.png)

4. Наполнение контентом
Страницы реализованы на языке разметки Markdown. В раздел «Проекты» добавлены кейсы по маркетингу (2ГИС) и технической поддержке (ИС «Наряды-допуски»). Для повышения интерактивности внедрены блоки admonition (заметки) с кнопками для скачивания презентаций.

![git_5](screenshots/sc_5.png)

5. Публикация (Deployment)
Публикация выполнена с использованием встроенного инструмента gh-deploy.
Команды:

git add . — индексация изменений.

git commit -m "Final version" — фиксация версии.

mkdocs gh-deploy --force — автоматическая сборка и деплой в ветку gh-pages.

![git_6](screenshots/sc_6.png)
